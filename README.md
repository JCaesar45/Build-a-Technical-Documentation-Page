# 🧾 Technical Documentation Page - Python Basics

This is a responsive technical documentation page built with HTML and CSS. It covers the basics of Python programming and fulfills all the user stories and tests required by the [freeCodeCamp Responsive Web Design Certification](https://www.freecodecamp.org/learn/).

## 📋 Features

- Semantic HTML5 structure
- Responsive layout with a sticky left-side navbar
- Media query support for mobile/tablet devices
- Internal navigation using anchor links
- Covers five sections of Python basics:
  - Introduction
  - Variables and Data Types
  - Control Flow
  - Functions
  - Modules and Imports

## 🛠️ Technologies Used

- HTML5
- CSS3 (including Flexbox and Media Queries)
- No external frameworks or libraries

## 🚀 How to Use

1. **Clone the repository or download the files:**

```bash
git clone https://github.com/your-username/technical-docs-page.git
cd technical-docs-page
```

2. **Open the `index.html` file in your browser:**

```bash
open index.html
```

Or just double-click `index.html` on your machine.

The layout adjusts for smaller screens using a media query:

* Navbar moves to the top
* Main content fills the screen below
